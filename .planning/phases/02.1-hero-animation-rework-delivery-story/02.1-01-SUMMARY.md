## Implementation Notes

**HeroDeliveryAnimation.tsx** implements three sequential phases:
- Phase 1: Day counter ticks 1→14 via `setInterval` at 130ms per tick (~1.8s total)
- Phase 2: Four browser-mockup sections snap in with `stagger(0.08)` via `useAnimate` (~0.5s)
- Phase 3: "Lansert!" badge pops in with scale keyframes `[0.8, 1.05, 1]` (~0.3s)
- 2.5s rest pause, then instant reset and loop

**Reduced-motion path** renders `StaticDeliveryDone` — plain HTML showing the completed "day 14" state with no `motion.*` wrappers and no timers. Handled internally so `HeroIsland` passes no props.

**HeroIsland.tsx changes** are minimal:
- Added `HeroDeliveryAnimation` import
- Removed `stats` array (48 lines → 6 lines for the right column)
- Both animated and reduced-motion paths now render `<HeroDeliveryAnimation />`
- All outer `motion.div` wrapper props on the right column preserved exactly
