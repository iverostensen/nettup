{
  messages: Array<{ role: 'user' | 'assistant'; content: string }>;

  <action>
Create `src/components/islands/ChatWidget.tsx` — a complete React island with these sections:

**Props and Types:**
- `interface ChatWidgetProps { currentPage: string }`
- `interface ChatMessage { role: 'user' | 'assistant'; content: string }`
- `interface LeadFormData { navn: string; epost: string; melding: string }`

**State:**
- `isOpen: boolean` — chat window open/closed
- `messages: ChatMessage[]` — conversation history (ephemeral, component state only)
- `input: string` — current input field value
- `isStreaming: boolean` — whether AI is currently responding
- `showTeaser: boolean` — teaser tooltip visible
- `showLeadForm: boolean` — in-chat contact capture form visible
- `leadSubmitted: boolean` — lead form was submitted successfully

**Teaser tooltip:**
- Auto-shows after 5 seconds via useEffect timer
- Text: "Lurer du pa noe?" in a small speech bubble above the chat bubble
- Dismisses when user clicks bubble (opens chat) or after 10 seconds
- Framer Motion fadeIn + slight translateY(-4px) animation
- Do NOT show teaser if chat is already open

**Chat bubble:**
- Fixed position bottom-right: `fixed bottom-6 right-6` (mobile: `bottom-4 right-4`)
- Z-index: `z-40` (below FloatingNav z-50)
- Circular button with chat icon (use an inline SVG speech bubble icon, no icon library)
- `bg-brand hover:bg-brand-light` with transition
- Size: 56px (w-14 h-14)
- When open, icon changes to X (close)
- Framer Motion scale animation on mount

**Chat window:**
- AnimatePresence for mount/unmount animation (scale + opacity from bottom-right origin)
- Desktop: fixed, 380px wide, 520px tall, positioned above the bubble (bottom-24 right-6)
- Mobile (< 640px): fixed, full width (inset-x-4), near full height (top-4 bottom-20), with close button in header
- `bg-surface-raised` background, `border border-white/10` subtle border, rounded-2xl
- Shadow: `shadow-2xl`
- Z-index: `z-[60]` (above FloatingNav when open, since user explicitly opened it)

**Chat window header:**
- "Nettup AI" title with small brand-colored dot indicator
- Close button (X icon) on the right
- `border-b border-white/10`

**Messages area:**
- Scrollable container (flex-1 overflow-y-auto)
- Auto-scroll to bottom on new messages via useEffect + ref
- Initial welcome message from assistant: "Hei! Jeg er Nettup sin AI-assistent. Hvordan kan jeg hjelpe deg i dag?"
- User messages: right-aligned, `bg-brand/20` background, `text-text`
- Assistant messages: left-aligned, `bg-white/5` background, `text-text`
- Simple text rendering — split on `\n` for line breaks, no markdown parser needed
- Typing indicator: three dots with staggered pulse animation (show when isStreaming and last assistant message is empty)

**Input area:**
- Text input + send button in a row at bottom
- `bg-surface` input with `border border-white/10`, `text-text`, `placeholder-text-muted`
- Placeholder: "Skriv en melding..."
- Send button: brand-colored arrow icon, disabled when input empty or streaming
- Submit on Enter key (not Shift+Enter which inserts newline)
- Clear input after sending

**Streaming logic:**
- `sendMessage` async function:
  1. Add user message to messages state
  2. Clear input, set isStreaming true
  3. Add empty assistant message placeholder
  4. POST to `/api/chat` with messages array (all messages for context) and currentPage
  5. Read SSE stream: parse `data:` lines, accumulate text into last assistant message
  6. On `[DONE]` or stream end: set isStreaming false
  7. On error: append error text to assistant message or show "Beklager, noe gikk galt. Prov igjen."
  8. Wrap in try/catch for network errors

**In-chat lead capture:**
- After the assistant's 3rd response (or when assistant mentions contact), show a subtle "Vil du at vi tar kontakt?" prompt below the messages
- When clicked, `showLeadForm` toggles a mini form inline in the chat:
  - navn (text input, required)
  - epost (email input, required)
  - melding (textarea, optional, pre-filled with conversation summary if possible)
  - Submit button: "Send"
- On submit: POST to `https://formspree.io/f/xnjnzybj` with `{ navn, epost, melding, kilde: 'chatbot' }`
- On success: set `leadSubmitted` true, show "Takk! Vi tar kontakt snart." message
- Also show a "Eller ga til kontaktskjemaet" link to /kontakt as alternative

**Accessibility:**
- `aria-label` on bubble button: "Apne chat"
- `role="dialog"` on chat window
- Focus trap when chat is open (focus input on open)
- `prefers-reduced-motion`: use `useReducedMotion()` from framer-motion, skip animations when true
- Escape key closes chat window

**Styling:**
- Use Tailwind 4 classes with brand tokens (bg-surface, bg-surface-raised, text-brand, etc.)
- Use `cn()` utility from existing codebase for conditional classes: `import { cn } from '@/lib/utils'` (check if this exists, otherwise inline clsx+twMerge)

TypeScript strict. No `any`. All event handlers properly typed.
