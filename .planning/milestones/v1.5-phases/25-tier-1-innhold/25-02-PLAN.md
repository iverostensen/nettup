  <action>
Inside `<BaseLayout title={city.metaTitle} description={city.metaDescription}>`, before `<main>`, add two `<script>` elements with `slot="head"` and `type="application/ld+json"`:

**Block 1 — Service schema:**
```astro
<script
  slot="head"
  type="application/ld+json"
  set:html={JSON.stringify({
    "@context": "https://schema.org",
    "@type": "Service",
    "name": `Nettside for bedrift i ${city.name}`,
    "provider": { "@id": "https://nettup.no/#business" },
    "areaServed": { "@type": "City", "name": city.name },
    "url": `https://nettup.no/steder/${city.slug}`
  })}
/>
```

**Block 2 — FAQPage schema:**
```astro
<script
  slot="head"
  type="application/ld+json"
  set:html={JSON.stringify({
    "@context": "https://schema.org",
    "@type": "FAQPage",
    "mainEntity": city.faq.map((item) => ({
      "@type": "Question",
      "name": item.question,
      "acceptedAnswer": { "@type": "Answer", "text": item.answer }
    }))
  })}
/>
```

**Key rules:**
- `provider: { "@id": "https://nettup.no/#business" }` — references the canonical LocalBusiness entity declared in BaseLayout. Never add a new LocalBusiness block.
- FAQ answers are stored as plain text in locations.ts (no HTML anchors) — `item.answer` is safe to use directly in `text` field.
- Place both script tags between the opening `<BaseLayout ...>` tag and `<main>`.
- Do NOT modify the existing template content inside `<main>`.
