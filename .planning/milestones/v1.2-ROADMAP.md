## Overview

Mål-først priskalkulator erstattet med fullverdig additiv prisestimator. Brukeren velger tjenestetype, størrelse, features, integrasjoner og designnivå — wizarden beregner en min/maks prisestimering med linjeoppstilling, 40% lanseringsrabatt og kontakt-CTA. Tilgjengelig som dedikert `/priskalkulator`-side og som seksjon på `/tjenester`.

## Milestone Summary

**Decimal Phases:**
- Phase 16.1: Inserted after Phase 16 as urgent UX simplification — per-item prices created decision friction

**Key Decisions:**
- Pricing config as single TS file, not JSON — type safety and IDE support
- `useReducer` with downstream reset on goal change — different services have different option sets
- Direction-aware slide animations (40px offset) — polished UX without heavy abstraction
- `/priskalkulator` not added to FloatingNav — tool page, not a top-level nav section
- Item rows show name only (no per-item prices) — focuses decision on total estimate

**Issues Resolved:**
- Per-item pricing on result screen created confusion — resolved with Phase 16.1 simplification

**Issues Deferred:**
- Monthly cost breakdown (additive per feature) — RES-FUTURE-01 for v1.3
- Running total updating in real-time — UX-FUTURE-01 for v1.3

**Technical Debt Incurred:**
- None significant — clean implementation, TDD coverage on engine
