## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, step number padding |
| md | 16px | Default element spacing, container inline padding (mobile) |
| lg | 24px | Section padding, card internal padding |
| xl | 32px | Layout gaps between content blocks |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level vertical padding (pt-16 = 64px for fixed header clearance) |

Exceptions: Thank-you page uses `pt-16` (64px) for fixed header clearance from `LandingPageLayout`, which already applies this via `<div class="pt-16">`.

Source: Established Tailwind spacing scale, `LandingPageLayout.astro` fixed header pattern.

## Component Inventory

This phase creates one new visual component (the thank-you page). All other work is config files, form behavior changes, and tracking wiring with no visual output.

### `/nettside-for-bedrift/takk` Page Layout

```
LandingPageLayout (existing)
  +-- Sticky header with logo + phone (from layout)
  +-- Main content area
  |     +-- Success icon (animated checkmark, cyan accent)
  |     +-- h1: "Takk for henvendelsen!"
  |     +-- Subtitle: "Vi kontakter deg innen 24 timer."
  |     +-- Next-steps card (bg-surface-raised, border-white/10, rounded-2xl)
  |           +-- h2: "Hva skjer na?"
  |           +-- Step 1: number circle + title + description
  |           +-- Step 2: number circle + title + description
  |           +-- Step 3: number circle + title + description
  +-- MinimalFooter (from layout)
  +-- Cookie consent banner (from layout)
  +-- Conversion tracking scripts (inline)
```

### Success Icon Spec

- Checkmark inside a circle
- Size: 64px (w-16 h-16)
- Circle: `bg-brand/10` with `border border-brand/20`
- Checkmark: `text-brand` stroke icon, stroke-width 2
- Animation: `animate-scale-in` from global.css (0.3s ease-out)
- Reduced motion: no animation, icon appears instantly

### Next-Steps Card Spec

- Container: `rounded-2xl border border-white/10 bg-surface-raised p-6 md:p-8`
- Max width: `max-w-2xl mx-auto`
- Steps use numbered circles: `w-8 h-8 rounded-full bg-brand/10 text-brand text-sm font-bold flex items-center justify-center`
- Step content: title in `font-bold text-text`, description in `text-text-muted text-sm`
- Vertical spacing between steps: 24px (`space-y-6`)
- Optional: thin vertical connector line between step circles using `border-l border-white/10`

### Form Redirect Interaction

- After Formspree returns 200: `window.location.href = '/nettside-for-bedrift/takk'`
- No loading spinner change needed (browser handles navigation)
- Button remains in submitting/disabled state until redirect completes (prevents double-submit)
- Context gate: redirect only when `context === 'b2b'`. Non-b2b forms keep existing inline success behavior.
