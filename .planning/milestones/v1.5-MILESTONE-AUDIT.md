### Flow 1: Organic discovery → city page → internal navigation ✓ Complete

Search result → `/steder/{slug}` (statically generated) → page renders with unique title/meta (SEO-02), Service JSON-LD (SEO-01), FAQPage JSON-LD (SEO-03), canonical URL (INFRA-04), intro copy (CONTENT-01), FAQ accordion (CONTENT-02) → nearbyAreas pill links to sibling cities (all 11 slug refs valid) → Footer "Områder vi dekker" column visible (LINK-01) → CTA navigates to `/kontakt` (LINK-02)

Wizard and chatbot events wired identically (ANAL-02).

### Flow 3: SPA navigation with persistent nav ✓ Complete (human-verified 2026-03-13)
